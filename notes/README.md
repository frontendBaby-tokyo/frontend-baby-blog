# Notes - Obsidian Task Management

This folder is dedicated to task management using Obsidian. It serves as a centralized workspace for organizing, tracking, and managing tasks and notes using the powerful features of Obsidian.

## Overview

This notes folder is designed to leverage Obsidian's capabilities for effective task management, including:
- Task creation and tracking
- Knowledge linking and cross-referencing

## Getting Started

### 1. Obsidian Setup
- Install [Obsidian](https://obsidian.md/) on your device
- Open this folder as an Obsidian vault
- Configure recommended plugins (see below)

### 2. Essential Plugins
- **Tasks**: Advanced task management with queries and filters
- **Calendar**: Visual calendar integration
- **Templater**: Dynamic template creation
- **Tag Wrangler**: Efficient tag management

## Task Management Features

### Task Syntax
```markdown
- [ ] Incomplete task
- [x] Completed task
- [!] Important task
- [?] Question or uncertain task
- [>] Scheduled/forwarded task
```

### Task Queries
Use the Tasks plugin to create dynamic task lists:
```markdown
```tasks
not done
due before tomorrow
```
```

### Tagging System
- `#work` - Work-related tasks
- `#personal` - Personal tasks
- `#urgent` - High priority items
- `#project/[name]` - Project-specific tags
- `#waiting` - Tasks waiting on others
